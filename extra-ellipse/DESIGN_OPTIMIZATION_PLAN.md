# Plan B Blog 设计优化任务计划

## 目标

让整站从“首页有鲜明收藏感、其他页面偏通用模板”升级为一套统一、克制、具有编辑气质的个人博客体验。

本轮不改变现有内容模型，不推翻唱片架与书架；它们继续作为整站的品牌基准。优化重点是 Blog 索引、文章阅读、订阅页与跨页面视觉一致性。

## 设计原则

1. **内容优先**：文章标题、摘要和阅读路径优先于筛选控件与装饰。
2. **编辑感而非后台感**：减少仪表盘式卡片、胶囊和复杂控制器。
3. **真实素材优先**：没有真实封面时不生成大块占位图。
4. **克制的一致性**：统一文字层级、线条、图标、交互状态和间距节奏。
5. **移动端先读后筛**：搜索与文章应先出现，次要筛选按需展开。
6. **渐进增强**：搜索和筛选依赖 JavaScript，但文章内容本身保持可访问。

- 文末导航具有足够点击面积，标题不会溢出。

## 暂不纳入本轮

- 邮件订阅后端与第三方服务接入。
- 深色模式。
- 新增第三类收藏展示组件。
- 大规模重写 About 文案。
- 更换整站字体或引入外部字体服务。

## 执行顺序

1. Blog 索引信息架构与响应式重构。
2. Subscribe 与 Emoji/图标一致性清理。
3. 文章阅读节奏与文末导航优化。
4. 构建、桌面与移动端视觉复核。
