# Plan B Blog 第二轮设计优化行动指南

## 目标

在不改变现有品牌基调、唱片架和书架交互的前提下，移除旧模板式组件残留，让首页、文章页与 About 页统一为更克制的编辑式语言。

## 设计原则

1. **一个页面只保留一个主要阅读入口**：减少重复的文章推荐模块。
2. **无真实素材时不制造视觉素材**：文章没有封面就不显示占位图。
3. **收藏组件是视觉主角，文章组件是内容主角**：两者不使用同一种卡片语言。
4. **移动端优先可扫描性**：缩短 About 文案，增加结构停顿。
5. **视觉精致不牺牲可用性**：触控区域不小于 40–44px，目录入口必须可理解。
6. **中英文有明确分工**：品牌主张和主题名可用英文，操作入口使用中文。

**验收标准**

- 单个相邻文章导航不再留下半屏空位。
- 移动端左右切换按钮具备至少 40px 的点击范围。

## 暂不调整

- Hero 主标题与当前页面宽度。
- Blog 列表页的信息架构。
- 唱片架与书脊展开交互。
- 文章正文宽度与整体字体组合。
- 深色模式和新的收藏组件。

## 执行顺序

1. 重构首页最新文章并移除 Featured。
2. 精简 About 文案与主题结构。
3. 完善文章目录、文末导航和触控区域。
4. 完成构建、桌面/移动端复核并部署上线。
